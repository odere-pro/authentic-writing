# Why Your Agent Keeps Forgetting

> TL;DR: Your agent isn't forgetful — you never wrote the context down. A rule file fixes that.

Agents lose context between runs, and it is frustrating. You explain the same conventions every morning like it's the agent's first day, because for the agent, it is.

---

## The real problem

The problem isn't memory — it's that the context only ever lived in your head. Nothing the agent can re-read tomorrow.

---

## The fix

Write the context down once, in a file the agent reads first. Start with the five things you'd tell a new hire on day one.
